# Personal Portfolio

A modern and responsive personal portfolio website showcasing my projects, technical skills, and experience as a Software Engineer.

## 🔗 Live Demo
[https://harshan-portfolio.onrender.com](https://harshan-portfolio.onrender.com)

## Features
- Clean and minimalist UI
- Responsive design for all devices
- Skills section with categorized technologies
- Contact section with social links
- Smooth animations and hover effects

## Built With
- HTML5
- Tailwind CSS
- JavaScript

## Sections
- Home
- Skills
- Projects
- Contact

## Featured Projects

### Argus
An API uptime monitoring platform that automatically checks endpoint availability and displays real-time health status on a centralized dashboard.

**Tech Stack:** Python, Flask, PostgreSQL, APScheduler, Tailwind CSS

---

### SpendWise
A secure expense tracking web application that helps users manage expenses, categorize spending, and visualize insights through an intuitive dashboard.

**Tech Stack:** Python, Flask, PostgreSQL

---

### Savora
A Chrome extension for saving, organizing, and managing browser tabs with persistent storage and light/dark mode support.

**Tech Stack:** HTML, CSS, JavaScript

## 💻 Installation

Clone the repository:
```bash
git clone https://github.com/harshanrepo/portfolio.git
```

Navigate to the project folder:
```bash
cd portfolio
```

Open `index.html` in your browser.
